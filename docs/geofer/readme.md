一、进入 duckdb

```sql
SELECT extension_name, installed, description
```

```sql
SHOW databases,
```


```sql
SELECT image_analyze(image, image_desc)

```

```sql
SELECT
	h.name,
	h.address,
  b.name,
	b.address,
	round( ( st_distance ( h.location, b.location ) / 0.0111 ) * 1000 ) AS distance
FROM
	haoke AS h,
	bank AS b
WHERE ST_X(h.location) <= ST_X(b.location)
ORDER BY
	distance
LIMIT 5;
```
